Introduction

The W3C Web of Things (WoT) seeks to counter the fragmentation of the IoT, making it much easier to create applications without the need to master the disparate variety of IoT technologies and standards. To realize this, WoT specifies some principles and technology building blocks that can be used to complement new or existing IoT ecosystems.
