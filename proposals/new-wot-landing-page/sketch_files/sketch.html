W3C Web of Things

The Web of Things (WoT) seeks to counter the fragmentation of the IoT by using and extending existing, standardized Web technologies. Providing metadata and re-usable technological building blocks, W3C WoT enables easy integration across IoT platforms and application domains.

  • Interest Group

    Practical evaluation, exploration, and outreach to other organizations.

  • Working Group

    Normative work on WoT building blocks. Work underlies W3C Patent Policy.

  • Getting Started

    Learn more through our whitepaper.

  • Developers

    Implementations, online Things, and tools.

  • Content orchestration

    Features needed to synchronize multiple media or non-media content together, on one or several devices.

  • Media capture

    Features needed to capture media content from a variety of available sources.

  • Media application development

    Platform features and best practices to develop media applications on the Web
